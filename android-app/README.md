# Android App

Native Android app for offline QR text transfer.

## Features

- Convert plaintext in textarea to QR code.
- Scan QR code from camera and write decoded text into textarea.
- Copy decoded/typed text to clipboard.
- Works offline at runtime.

## Build & Run

1. Open this `android-app/` folder in Android Studio.
2. Sync Gradle.
3. Run on an Android device (camera needed for scan).

## Permissions

- `CAMERA`: used only for QR scanning.
