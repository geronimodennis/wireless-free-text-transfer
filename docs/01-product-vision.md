# Product Vision

## Problem Statement

Users need a quick, offline, and private way to transfer plaintext between desktop computers (Mac/Windows) and Android devices without network setup, cloud services, or internet connectivity.

## Vision

Deliver a local-first QR transfer app pair where either device can:

- Convert text to QR code.
- Read QR code and write decoded text into a text area.

## Product Goals

- Zero server dependency at runtime.
- Zero internet dependency at runtime.
- Fast transfer for short-to-medium plaintext notes.
- Simple UX for non-technical users.

## Non-Goals (MVP)

- File transfer.
- End-to-end encrypted payload protocol.
- Multi-device sync/history storage.
- Continuous camera scanning in background.

## Success Metrics

- Decode success rate > 95% for normal lighting.
- Average transfer completion < 10 seconds for text under 1 KB.
- Crash-free session rate > 99% during pilot.
